This is the top of the Network Inspired Transportation System simulator. 

NITS contains the virtuanenv environment files.
NITS_CODE contains the code
NITS_CODE/hermes contains the views
NITS_CODE/NITS_CODE/site_media contains the javascript files
NITS_CODE/templates contains the html files

to start the simulator.
1) activate the environment . <PATH>/NITS/bin/activate
2) start the test server from <PATH>/NITS_CODE and run "python manage.py runserver 0.0.0.0:8888"
3) direct any browser to localhost:8888
